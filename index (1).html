Git

Primeiro passo a se fazer é a configuração de usuario git config --global user.name "(nome do usuario)" TS

Segundo passo configuração do email a ser usado git config --global user.email "(email do usuario)"

Apos a configuração voce pode modificar o arquivo, ele passara para modificado(M) deve se usar o comando Git add (nome do arquivo/repositorio) para ir para o estado de Modific Stage(S) ai para poder usar o comando git commit -m "A mensagem da alteração"

Caso queira ver a linha modificada do arquivo/ repositorio git diff e caso queira ver todas as alterações salvas (commits ) é só usar o comando git log e para restaurar alguma modifição finalizada(commit) usase o comando git restore (nome do arquivo)
